# Rubik's Cube

A 3D Rubik's Cube playable in the browser — built with React and Three.js.

**[Try it live →](https://ogorodnikoff2012.github.io/rubiks-cube/)**

## Controls

### Moves

| Key       | Move           |
| --------- | -------------- |
| `f` / `F` | Front CW / CCW |
| `b` / `B` | Back CW / CCW  |
| `r` / `R` | Right CW / CCW |
| `l` / `L` | Left CW / CCW  |
| `u` / `U` | Up CW / CCW    |
| `d` / `D` | Down CW / CCW  |

Lowercase = clockwise, uppercase (Shift) = counter-clockwise.

### Other

| Key                       | Action           |
| ------------------------- | ---------------- |
| `Ctrl+Z`                  | Undo             |
| `Ctrl+Shift+Z` / `Ctrl+Y` | Redo             |
| `Escape`                  | Clear move queue |

**Mouse:** click and drag to rotate the cube.
